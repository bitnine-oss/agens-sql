2.2. 특장점

  1. Open Source DBMS

    PostgreSQL은 현재 Open Source DBMS중에서 Enterprise급의 성능과 기능을 제공 하고 있고, 앞으로도 더욱 발전가능성이 무궁무진하다. 또한 사용자가 직접 커뮤니티에 참여하여 불편한 점이나 버그 리포트 등을 할 수 있다는 장점이 있다.

  2. 다양한 커뮤니티

    PostgreSQL의 장점 중 하나는 커뮤니티가 다양하다는 것이다. PostgreSQL이 Open Source DBMS인 만큼 사용자들이 원하는 모듈을 직접 개발 또는 커뮤니티에 건의 함으로써 만들 수 있다. 이러한 모듈이나 PostgreSQL 자체의 커뮤니티들은 많은 사람들의 의견을 취합하여 발전할 수 있다. 집단 지성의 이념처럼 다양하고 많은 커뮤니티가 발달하여 지식의 쉬운 전달이 가능하다.

  3. 함수

    ‘저장 프로시저’ 라고 불리는 SQL문으로 작성된 함수를 서버환경에서 사용할 수 있다. 또한 오라클의 PL/ SQL과 유사한 PL/pgSQL, PL/Python, PL/Perl, C/C++, PL/R 등 다양한 언어들을 지원하고 있다.

  4. ACID와 Transaction

    PostgreSQL은 ACID(원자성, 일관성, 고립성, 지속성)을 지원한다.

  5. 다양한 인덱스 기법

    PostgreSQL은 B+ tree 인덱스 기법 뿐만 아니라 GIN(Generalized Inverted Index), GiST(Generalized Search Tree)등 다양한 인덱스 기법을 제공한다.
