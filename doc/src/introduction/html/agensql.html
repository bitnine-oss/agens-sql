장 2. Agens SQL(아젠스) 란?

차례
2.1. 시스템 구조
2.2. 특장점
2.3. 확장성
2.4. 패키지

Agens(아젠스)는 라틴어로 고대 로마 황제의 정보보좌관을 뜻한다. 정보를 제공함에 있어서 필요한 정보를 가장 정확하고 빠르게 전달하고, 이러한 정보를 수집, 가공하여 최적의 정보를 제공하고자 하는 취지에서 만들어진 비트나인의 데이터베이스 통합 솔루션의 명칭이다.오픈소스 DBMS인 PostgreSQL과 그 확장 모듈에 대한 국내 최초의 통합 데이터베이스 패키지이다. Agens SQL1.0은 PostgreSQL9.4를 기반으로 하고 있으며, 누구나 무료로 사용할 수 있는 안정성 있고 검증된 통합 데이터베이스 패키지이다. 주요 확장 모듈로는 Query optimizer(pg_hint_plan), Connection pooling(PGBouncer), Event Processing System(PGQ), Remote Function Call(PL/Proxy), GIS System(PostGIS) 등이 있다.
