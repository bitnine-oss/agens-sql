1.2. 일러두기

이 문서는 다음과 같은 표기 규칙을 따른다: 대괄호 ([, ])는 선택 항목을 뜻한다. (Tcl 명령어에서는 물음표(?)로 대신한다.) 중괄호 ({, })와 세로 막대 (|)는 '이들 중 하나'를 선택해야 한다. 온점들 (...) 들은 해당 사항이 반복될 수 있음을 의미한다.

예제에서 SQL 명령어들은 => 프롬프트를 사용하며, 쉘 명령어들은 $ 프롬프트를 사용해서 구분하고 있다. 프롬프트가 없는 경우는 쉘 명령어를 뜻한다.

문서에서 관리자라고 언급하면 이는 대부분 서버 프로세스를 실행할 수 있는 권한이 있는 계정을 뜻하고, 사용자라고 언급하면 Agens-SQL 시스템을 사용하는 계정을 뜻한다. 이 문서에서 대체로 위와 같은 의미로 사용되고는 있지만, 그렇지 않은 부분도 있다.
