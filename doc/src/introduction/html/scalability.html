2.3. 확장성

Agens SQL은 유용한 오픈소스 모듈들을 선택적으로 사용가능한 형태로 제공함으로써 편리성과 안정성을 제공한다. 또한, Oracle의 오픈소스 버전이라고 평가를 받는만큼 기존의 Oracle 사용자들도 큰 이질감 없이 쉽게 사용할 수 있다. ACID 트랜잭션 바탕의 데이터의 정합성 제공, Replication, Failover, Connection Pooling 등의 고가용성 모듈 제공, SSL, Kerberos 등의 보안 모듈 제공 등 상용 DBMS 못지 않은 기능과 그 안정성을 제공한다.
