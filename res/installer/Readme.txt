Agens SQL V1.0.0 Enterprise Edition

 Agens SQL은 오픈소스 DBMS인 PostgreSQL과 그 확장 모듈에 대한 국내 최초의 통합 데이터베이스 패키지입니다.
 Agens SQL V1.0은 PostgreSQL 9.4를 기반으로 하고 있으며, 누구나 무료로 사용할 수 있는 안정성 있고 검증된 통합 데이터베이스 패키지입니다.
 Agens SQL은 Enterprise환경에서 활용가능한 강력한 오픈소스 모듈들을 제공하고 선택적으로 사용 가능하게 함으로써, 편리성과 안정성을
보장합니다. Agens SQL 제공하는 패키징을 활용하면 어떠한 확장 모듈도 즉각적인 사용이 가능하게 되며, Agens SQL은 고객지원과 오픈소스 커뮤니티를 통해 지속적으로 추가 모듈들이 선별되어 패키지에 포함되게 됩니다.

Agens SQL V1.0.0 Enterprise Edition에서 제공하는 패키지의 내용과 버젼은 다음과 같습니다.
 - postgresql-9.4.4
 - pgpool-II-3.4.2
 - skytools-3.2
 - pg_hint_plan94-1.1.3
 - plproxy-2.5
 - slony1-2.2.4
 - pgbouncer-1.5.5
 - postgis-2.1.7
 - powa-archivist-REL_2_0_0
